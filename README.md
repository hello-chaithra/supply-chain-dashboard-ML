# Supply Chain Dashboard using ML 
## Project Overview
ML-Driven Supply Chain Analytics Dashboard (Completed July 2025) 
- Used Python (Pandas, NumPy), R, SQL, Streamlit, Tabeau, Excel.
- Processed & analyzed **~10k+ records** from the public **DataCo Smart Supply Chain** dataset
- Developed interactive dashboard for inventory forecasting, demand analysis, and visualization using public dataset.
- Incorporated basic ML for trend prediction, Developed an **end-to-end interactive dashboard** & Deployed publicly.
